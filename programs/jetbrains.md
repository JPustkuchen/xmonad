
# JetBrains IDE

## a) Install wmname
```shell
sudo apt install suckless-tools
```

## b) Install IntelliJ Idea
Install, Unzip and define in `.bash_aliases`:
```shell
alias idea='wmname LG3D && IDEA_JDK=/usr/lib/jvm/java-8-oracle ~/programs/idea/bin/idea.sh &> /dev/null &'
```
